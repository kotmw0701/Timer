# Timer
Java言語で作成した、複数管理可能なミリ秒単位のタイマー

#### ショートカットキー(別ウィンドウ時)
F1: UI非表示<br>
F2: タイトル非表示<br>
F3: フルスクリーンにする<br>
F4: none<br>
F5: 一時停止<br>
F6: 開始<br>
F7: 停止<br>
F8: none<br>
F9: タイトル編集<br>
F10: 背景色変更<br>
F11: none<br>
F12: none<br>
